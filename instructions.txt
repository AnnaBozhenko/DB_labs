# Лабораторна робота 2: Міграція
        1. Створіть в директорії app директорію data та завантажте туди всі csv-файли з 2016 по 2021 роки
        2. створіть в директорії app/flyway директорію sql 
        3. запустіть в терміналі команду docker compose up app db
        4. дочекайтеся завершення роботи додатку
        5. запустіть в терміналі команду docker compose flyway
        6. моніторте стан бази:
                6.1 командою docker exec -ti <db image id> sh
                6.2 у відкритому shell контейнера бази введіть: psql -U Student -d ZNO
                6.3 вводьте необхідні команди для моніторингу стану бази, наприклад проглянути всі таблиці: \dt
